CSV to markdown table?

You may have to add a lot of computational results in your paper, markdown table is not designed for this. Use the filter pandoc-csv2table

With the csv data prepared, just include in the markdown doc:

{.table caption="This is the **caption**" source="table.csv"}

JS rendered html?

I use JS libraries (D3.js, mermaid) on some of the flowcharts.

Use the following html snippets in markdown.

```{=html}
<div class="mermaid">
  graph LR
      A --- B
      B-->C[fa:fa-ban forbidden]
      B-->D(fa:fa-spinner);
</div>
graph LR; A --- B; B-->C[fa:fa-ban forbidden]; B-->D(fa:fa-spinner);

Also include in the YAML metadata

---
# ...
header-includes:
  - |
    ```{=html}
    <script src="https://cdn.jsdelivr.net/npm/mermaid@8.4.0/dist/mermaid.min.js"></script>
    <script>mermaid.initialize({ startOnLoad: true });</script>
    ```
---

summarize

then a typical call can be:

~/.cabal/bin/pandoc \
  --filter pandoc-citeproc \
  --filter pandoc-sidenote \
  --filter pandoc-csv2table \
  --katex --toc \
  --csl=assets/ieee.csl \
  --css=assets/tufte.css \
  -o ${DOC}.html \
  meta.yaml -s ${DOC}.md
